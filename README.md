## [`Notes React Demo`](https://devilzer.github.io/notes-app/)

## Features

* Add new Note.
* Notes edit.
* Alert notifications.
* Persistant redux state.
* Search based notes filter.

## How to install and run?

1. Clone the repositories.
2. Install all dependencies by `npm install`
3. `npm start`
